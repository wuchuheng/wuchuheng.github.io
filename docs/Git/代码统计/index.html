09.代码统计

代码统计

$ git log --author="wuchuheng" --pretty=tformat: --numstat | awk '{ add += $1; subs += $2; loc += $1 - $2 } END { printf "added lines: %s, removed lines: %s, total lines: %s\n", add, subs, loc }'

结果输出

added lines: 11980, removed lines: 58, total lines: 11922
