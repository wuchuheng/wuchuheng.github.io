## 1 `SpringBoot`用 `Gradle` 打包

在项目的根目录下用命令行输入: 
``` bash 
$ gradle build
```
然后会在项目的`build > libs > **-SNAPSHOT.jar`就是了, 如： 
``` text {13}
.

├── META-INF

│   │   └── demojava-package-demo-0.0.1-SNAPSHOT.jar       // <-- 这个就是打包出来的包
│   ├── repojrts
│   │   └── tests
│   ├── resources

│   ├── test-results
│   │   └── test
```

然后直接运行这个`jar`包就可以了
``` bash
$ java -jar demojava-package-demo-0.0.1-SNAPSHOT.jar
```
