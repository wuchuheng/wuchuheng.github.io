## 1 定义dart的异常类

> 其实就是实现内置的`Exception`,就是自定义异常类了。

```dart
class RoutingFormatErrorException implements Exception {
  late String _message;
  RoutingFormatErrorException(
      [String message = 'Routing format error exception']) {
    _message = message;
  }

  @override
  String toString() {
    return _message;
  }
}



```
