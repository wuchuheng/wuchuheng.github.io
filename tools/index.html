19 线上IDE​

tip

线上IDE用于代码示例效果非常好，相对于那些直接展示代码片段来说，通过线上编辑器示例出来的代码更生动更好理解,比如一个算法示例:



常用的线上IDE有:

codepen →

codepen是很经典的代码示例网站，其中聚集世界web天才和一流的web作品,

stackblitz →

stackblitz 是个服务端和前端一锅端的线上编辑器，不仅提供html js css 的编辑能力，还提供node有后端服务能力，完全能进行前端工程化开发

codesandbox →

codesandbox 跟stackblitz一样是个能进行web工程化开发的线上IDE，用它用示例代码时，可以在同一套代中使用git分支能力从而同一示例代码中展示不同的示例, 好用，就是不太稳定

replit →

replit 跟stackblitz一样是个能进行web工程化开发的线上IDE，但它支持的开发语言更多,是目前比较全面的线上开发工具，适合在线演示和教学工具
